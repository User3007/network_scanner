🚀 Quick Start

1

Install Prerequisites

Download and install Node.js and Nmap on your system

2

Run Setup

Execute the setup script to install all dependencies

npm install && cd client && npm install
3

Start Application

Launch the development server and open your browser

npm run dev

⚠️ Important Security Notice

This tool performs network scanning which may be detected by security systems. Only scan networks you own or have explicit permission to scan. Be aware of your organization's network policies.
